リネームマップ

「変換前 → 変換後」形式（= や -> も可）。リネーム後の名前で全集計を行います。

タグ辞書（意味を追加登録）

既存タグと同名で登録すると意味を上書きします。登録タグは命名規則ガイド・CP名解析・⑥の色分け（紫）に反映されます。

命名規則ガイド — CP名の読み方・タグの意味・解析ツール

CP名の構造（例）

三木_728_YT_NST_メンズクリア_mktag_mc_YT_NST10v5_brd_7m
管理用プレフィックス（担当者・番号・媒体・案件。集計では使わない） mktag = 区切りマーカー（これ以降を解析対象にする） NST10v5 = シリーズ10・バージョン5（①は数字のみで大括り、③はv単位） ターゲティング系タグ（brd / 年齢 / 市区切り） yoko = 横動画（なければ縦動画） ユーザー登録タグ（タグ辞書）

タグの意味一覧

タグ意味集計への影響
brdブロード配信② brdグループ（最優先。ただし年齢タグ併記でyokoなしの場合は年齢が優先）
18-24 / 25-34 / 35-44 / 45-54年齢セグメント② 年齢セグメントグループ
6m / 7m など月限定（6月・7月…）② 市区切りグループ判定に使用
st / kb / ks / yh / fo / hs / kt / spさいたま / 神戸 / 川崎 / 横浜 / 福岡 / 広島 / 京都 / 札幌② 市区切りグループ
1 / 2 / 101 など（数字のみ）市区切り番号② 市区切りグループ（後ろに _new が付いても同様）
yoko横動画④ 横動画（タグなし = 縦動画）
new再入稿影響なし
nare / os / ng特殊タグ影響なし（除外もしない）
wareki和暦影響なし

振り分けの優先順位：brd → 年齢セグメント → 市区切り → その他（brd + yoko + 年齢 は brd、brd + 年齢のみは年齢セグ）。「設定 → タグ辞書」で登録したタグは紫色でこの表の末尾に追加されます。
